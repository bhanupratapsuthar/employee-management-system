## 📌 Overview

The **Employee Management System (EMS)** is a web-based application designed to manage employees and their tasks efficiently. It provides role-based access where an **admin** can assign and manage tasks, and **employees** can track their assigned work.

This project demonstrates modern frontend development using React with a focus on state management, UI design, and task handling logic.

---

## 🚀 Features

* 🔐 Role-based authentication (Admin & Employee)
* 📋 Task assignment and tracking
* 📊 Task status overview:

  * Active Tasks
  * New Tasks
  * Completed Tasks
  * Failed Tasks
* 👨‍💼 Admin dashboard to manage employees
* 👩‍💻 Employee dashboard to view and update tasks
* 🎨 Clean and responsive UI

## 📂 Project Structure (Overview)

* `components/` → Reusable UI components
* `context/` → Global state management (Auth & Data)
* `pages/` → Admin & Employee dashboards
* `data/` → Static employee and task data

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **State Management:** Context API
* **Deployment:** Vercel

---

## 💡 How It Works

* Admin logs in and assigns tasks to employees
* Employees log in and:

  * View assigned tasks
  * Track progress
  * Mark tasks as completed or failed
* Dashboard dynamically updates task counts and statuses
